What is the difference between Margin, Border and Padding

CSS is the mark up language that web developers use to style items, or elements, on a web page.

When placing elements on a page, we use the the properties Padding and Margin to tell the elements where we want themt o sit.

  • Padding describes the space between the content of the element and its border.
  • Margin describes the space between the border of the element and the next element.

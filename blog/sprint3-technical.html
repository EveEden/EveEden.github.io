The Languages of the World Wide Web

Use an analogy to describe the differences between HTML and CSS

HTML and CSS are both used to create web pages, in a similar way that we would create a poster.

HTML is like the blank paper with a pencil draft on it, it would be like a basic outline of all the elements you wanted to put on the page.

CSS is like the paint and ink you use to create the background colours, the pictures and fonts styles and sizes. Its what makes the page pretty.

Explain control flow and loops using an example process from everyday life

When you drive to work in the morning, there is a particular set of things you do, and responses from the car, that get you to your destination.

A similar process is flow control and looping when executing code.

Turning the key and putting the car into gear is like giving an input to the engine to execute a pre-determined set of steps - If the key turns, flick a switch to the starter motor and start the engine.

As with code, inputs can be variable - you can put the car into reverse, or first, or let it idle while you turn the heater. The inputs that are chosen tell the electronics in the car to execute certain steps - IF the user puts the car to reverse, move the car, ELSE idle the car. This would be an If/Else statement if our cars understood code.

Explain the difference between accessing data from arrays and objects.

An Array is a list of 'things', usually strings. As you can imagine, it has a length - for example, a list of 3 things. You can access data from it by calling the list and asking for an item in a certain position, such as the second item.

I like to think of an object like everyday objects - such as a basketball, which has properties like its colour, and the property has a value, in this case orange. To call data from objects, we access the property, like colour, which returns the value. In the case of the basketball, the value returned would be orange.

Explain what functions are and why they are useful.

Functions are basically a set of instructions written in code. The computer executes these when the function is called. A function that is the property of an object is called a Method.
